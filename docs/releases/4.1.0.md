# Release Notes – 4.1.0

## 简体中文 (zh-CN)

- 新增自定义词表功能，可创建、编辑、启用、导入和导出词表，用于提升特定词汇的语音识别纠错效果。
- 词表支持词条启用、删除、排序，并显示词条数量和有效字符使用情况。
- 新增全局开始 / 停止识别快捷键。
- 新增应用内公告提示，支持关闭或不再显示。

## English (en)

- Added custom vocabulary tables for creating, editing, enabling, importing, and exporting terms to improve recognition correction for specific words.
- Vocabulary entries can now be enabled, deleted, reordered, and tracked with entry count and effective character usage.
- Added a global start / stop recognition hotkey.
- Added in-app announcements, with options to dismiss or hide them permanently.

## 日本語 (ja)

- カスタム用語集を追加しました。用語集の作成、編集、有効化、インポート、エクスポートに対応し、特定語句の音声認識補正を改善できます。
- 用語エントリの有効化、削除、並べ替えに対応し、エントリ数と有効文字数も確認できます。
- 認識の開始 / 停止を切り替えるグローバルホットキーを追加しました。
- アプリ内お知らせを追加し、閉じる、または今後表示しない設定ができるようになりました。
