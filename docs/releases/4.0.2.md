# Release Notes – 4.0.2

## 简体中文 (zh-CN)

- **breaking:** 调整服务接入方式，客户端现统一使用官方服务，自定义服务器功能下线，同时官方提供的 URL 和 Key 不再可用。
- **feat:** 未订阅用户现在拥有按时间刷新的免费额度，主页面会显示当前免费额度状态；未登录时无法开始识别，免费额度用尽时也会给出提示。
- **feat:** 新增应用内公告提示，重要通知会直接显示在顶部，也可以选择关闭或不再提示。
- **feat:** 账号页新增余额与自动续费信息，可查看余额、进行充值，并管理自动续费状态。
- **improve:** 优化了登录、订阅和账号信息相关提示，减少使用过程中的困惑。

## English (en)

- **breaking:** Updated the service access model. The client now uses the official service only, custom server support has been removed, and the previously provided official URL and Key are no longer available.
- **feat:** Non-subscribed users now receive a free quota that refreshes over time. The home page displays the current free quota status; recognition cannot start while logged out, and users will also see a prompt when the free quota is exhausted.
- **feat:** Added in-app announcement prompts. Important notices are now shown directly at the top of the app, and users can close them or choose not to see them again.
- **feat:** Added balance and auto-renewal details to the account page, including balance viewing, top-up entry points, and auto-renewal management.
- **improve:** Refined login, subscription, and account-related prompts to reduce confusion during use.
